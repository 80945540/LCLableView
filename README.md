## 使用说明

导入 lclablelibrary 到项目中

在 build.gradle 的 dependencies 添加:
```
	dependencies {
    compile fileTree(include: ['*.jar'], dir: 'libs')
    ....
    compile project(':lclablelibrary')
}
```
